# Система учёта оборудования и складов

## Описание проекта
Веб-приложение для учёта оборудования на складах. Реализованы CRUD операции для складов и оборудования, поиск оборудования, пагинация.

## Архитектура
**Выбран вариант А** (раздельные фронтенд и бэкенд):
- Frontend: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- Backend: Express + TypeScript
- Хранилище: In-memory с сохранением в JSON-файл

## Стек технологий
- Next.js 14
- TypeScript
- Tailwind CSS
- Express
- Zod (валидация)
